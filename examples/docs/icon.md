# Icon 图标
----
### 基础用法
直接通过设置类名为 `ix-icon-iconName` 来使用即可。例如：

<div class="demo-block">
  <i class="ix-icon-sound-Mute"></i>
  <i class="ix-icon-sound-filling"></i>
  <i class="ix-icon-stop"></i>
  <ix-button type="primary" icon="ix-icon-search">搜索</ix-button>
</div>

::: demo
